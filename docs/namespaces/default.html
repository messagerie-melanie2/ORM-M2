MODE_SIMPLE ou MODE_MULTIPLE pour la configuration TYPE_EXTERNAL Le MODE_SIMPLE va lire les donnes directement dans le CONFIGURATION_PATH Le MODE_MULTIPLE permet de grer plusieurs configuration dans le CONFIGURATION_PATH_LIBM2 Dans ce cas la configuration va tre lu dans le rpertoire correspondant au CONFIGURATION_APP_LIBM2 qui doit tre configur dans l'application
